# BBF Brand Icons

Iconos custom Brand Brain Foundry (NO Lucide React).

**Reservado para M6+.** Lucide cubre la mayoría de necesidades UI actuales.

Usar para: símbolos brand-specific que no existen en Lucide library.

## Atom relacionado

`src/components/atoms/Icon/` (wrapper Lucide React)

Para iconos custom BBF, crear `src/components/atoms/CustomIcon/` con loader pattern similar a BBFLogo.

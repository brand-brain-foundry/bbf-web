# BBF Open Graph Images

Imágenes Open Graph (social media previews) por página y locale.

## Convenciones

- `og-{page}-{locale}.png` (ej: `og-home-es.png`, `og-home-en.png`)
- Tamaño canon: 1200x630px (Facebook/LinkedIn/Twitter)
- Format: PNG (compatibilidad universal)

## Default fallback

`og-default.png` — usado cuando no hay og específico por página.

## Páginas con OG específico (M6+)

- `og-home-es.png`, `og-home-en.png` (M5-D5+)
- `og-about-es.png`, `og-about-en.png` (M6+)
- `og-cases-es.png`, `og-cases-en.png` (M6+)
